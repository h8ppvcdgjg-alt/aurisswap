AurisSwap supports early-stage tokens with a clear, fair, and sustainable listing path. We prioritize real liquidity, transparent tokenomics, and long-term participation.

V1

Why list on AurisSwap?

Real Liquidity

Designed to discourage short-term extraction and reward long-term holders.

AurisCoin Alignment

AurisCoin acts as a core ecosystem asset across key flows and incentives.

Simple Process

Fast pool creation and clear requirements to reduce spam and scams.

Pool Creation Fee

Creating a new pool requires a fixed fee in AurisCoin. This strengthens AurisCoin utility and filters spam listings.

Exact amounts may be updated as the protocol evolves. See Fees for the current schedule.

Basic Requirements (V1)

  • Verified contract address (no proxies that can change supply without disclosure)
  • Clear token symbol, decimals, and supply information
  • Project description + official links (website / X / Telegram / Discord)
  • No obvious honeypot / blacklist / malicious transfer restrictions

Note: AurisSwap may request additional information for security and ecosystem integrity.

How to apply

Send the following details to the AurisSwap team:

  • Token name + symbol
  • Contract address
  • Chain (BNB Chain)
  • Website + socials
  • Short description (1–3 paragraphs)
  • Liquidity plan (initial LP size and timing)

(We’ll add an application form on this page in a future update.)
